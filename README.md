# Bot Simsimi Perl

## require

- `perl`

## install perl

```bash
sudo pacman -S perl
```

## install modul

```bash
cpan install LWP::UserAgent JSON
```

```bash
cpan install LWP::Protocol::https
```

note: kalau gak bisa install (instalasi gagal), jalankan perintahnya sebagai root.

## jalankan

```bash
perl app.pl
```
